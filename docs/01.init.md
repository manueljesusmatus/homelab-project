# Configurar raspberry
Guía para configurar y levantar los servicios iniciales en raspberry.

Instala dependencias del sistema, pipx, Ansible y las colecciones requeridas.

```bash
curl -sSL https://raw.githubusercontent.com/manueljesusmatus/homelab-project/master/scripts/init.sh | bash
```

---

### Pasos manuales (referencia)

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install pipx git -y
pipx ensurepath
export PATH=$PATH:$HOME/.local/bin

pipx install --include-deps ansible
pipx inject ansible requests docker

ansible-galaxy collection install community.docker --force
ansible-galaxy collection install ansible.posix --force
```
