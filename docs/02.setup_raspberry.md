# Setup raspberry

## Instalar dependencias

```bash
curl -sSL https://raw.githubusercontent.com/manueljesusmatus/homelab-project/master/scripts/init.sh | bash
```

> Instala git, pipx, Ansible y colecciones requeridas. Ver [01.init.md](01.init.md) para detalle.

---

## Configurar Repositorio de Plantillas

1. Generar una clave SSH para autenticación y agregarla a GitHub:
    ```bash
    ssh-keygen -t ed25519 -C "server@mmatush.cl"
    ```

2. Clona el repo:
    ```bash
    git clone git@github.com:manueljesusmatus/homelab-project.git ~/repos/lab/homelab-project
    ```

3. Crear `all.yml` con las variables requeridas (ver `all.example.yml`):
    ```bash
    nano ~/repos/lab/homelab-project/ansible/group_vars/all.yml
    ```

4. Ejecutar init de playbook
    ```shell
    ansible-playbook ~/repos/lab/homelab-project/ansible/lab_playbook.yml --tags "raspberry" --ask-become-pass
    ```

5. Reiniciar
    ```shell
    sudo reboot now
    ```

6. Levantar servicios docker
    ```shell
    ansible-playbook ~/repos/lab/homelab-project/ansible/lab_playbook.yml --tags "containers_up"
    ```

7. Establecer backups
    ```shell
    ansible-playbook ~/repos/lab/homelab-project/ansible/lab_playbook.yml --tags "containers_backup" --ask-become-pass
    ```

8. Restaurar volumes docker
    ```shell
    ansible-playbook ~/repos/lab/homelab-project/ansible/lab_playbook.yml --tags "containers_restore" --ask-become-pass
    ```

### Configurar RAM tracking [issue](https://forums.raspberrypi.com/viewtopic.php?t=203128)

Gestionado por Ansible, se ejecuta con el tag `raspberry` o `mem_issue`.
Agrega `cgroup_enable=cpuset cgroup_enable=memory cgroup_memory=1` a `/boot/firmware/cmdline.txt` y reinicia el equipo automáticamente si hubo cambios.
